**Консольная информационная система** для автоматизации учёта аренды оборудования.

## Описание проекта

Система предназначена для автоматизации бизнес-процессов компании, предоставляющей оборудование во временное пользование (аренду).

**Цель создания:**  
Повышение точности расчётов, контроль занятости активов и прозрачность всех арендных операций.

### Основные возможности
- Полноценное управление каталогом оборудования (CRUD)
- Работа с клиентами (регистрация, история, статус благонадёжности)
- Оформление и закрытие договоров аренды
- Автоматический расчёт стоимости и штрафов за просрочку
- Бронирование оборудования с проверкой доступности по датам
- Защита от двойного бронирования
- Управление персоналом
- Аналитика и отчётность

---

## Технологии
- **Язык программирования:** Python 3
- **База данных:** SQLite3 (файловая)
- **Архитектура:** Модульная (models, ui, database)
- Без внешних зависимостей

---

## Установка и запуск

1. Перейдите в папку с проектом
2. Выполните команду:

python main.py

При первом запуске автоматически создастся файл базы данных rent_accounting.db.
